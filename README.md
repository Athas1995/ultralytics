### Install

```bash
git clone https://github.com/ultralytics/ultralytics
cd ultralytics
python -m pip install --upgrade pip wheel
pip install .  # (dev)
# pip install ultralytics (production)
```


### Usage
```python
import ultralytics
from ultralytics import HUB, YOLO

ultralytics.checks()
```
